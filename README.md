# Personal Expense Tracker

Flask web app to track daily expenses and view total spending.

## Features
- Log expenses by category: Food, Transport, Data, Other
- View total amount spent
- Expense history with date + notes
- SQLite database

## Tech Stack
Python, Flask, SQLite, HTML

## Run it
1. pip install -r requirements.txt
2. sqlite3 expenses.db < schema.sql
3. python app.py
4. Open http://localhost:5000
